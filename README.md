# SchemeSync Navigator

An AI-powered government scheme discovery and application platform designed to help users identify, track, and apply for welfare schemes based on eligibility, profile data, and personalized recommendations.

This platform aims to bridge the gap between citizens and government welfare programs by simplifying access, improving awareness, and making scheme discovery smarter.

---

## Overview

Many users miss beneficial government schemes because of lack of awareness, complex eligibility criteria, and poor accessibility.

SchemeSync Navigator solves this by providing:

- Smart scheme recommendations
- Eligibility-based filtering
- Personalized dashboards
- Application tracking
- Admin-controlled scheme management

---

## Features

### User Features
